# RAYNO
## Описание проекта
__Rayno__ - сайт, посвящённый кино. Пользователь может оценивать фильмы и писать к ним рецензии. У всех фильмов автоматически формируется рейтинг на основе всех поставленных оценок. К рецензиям пользователи могут оставлять свои комментарии.
## Как установить и запустить проект
Клонировать репозиторий и перейти в него в командной строке:
~~~
git clone git@github.com:MrIvensZ/rayno.git
~~~
~~~
cd rayno
~~~
Cоздать и активировать виртуальное окружение:
~~~
python -m venv venv
~~~
~~~
source venv/bin/activate
~~~
Установить зависимости из файла requirements.txt:
~~~
python -m pip install --upgrade pip
~~~
~~~
pip install -r requirements.txt
~~~
Выполнить миграции:
~~~
python manage.py migrate
~~~
Создайте суперпользователя:
~~~
python manage.py createsuperuser
~~~
Запустить проект:
~~~
python manage.py runserver
~~~
## Начало работы
Когда проект запущен, перейдите в админ зону Django через бразуер:
~~~
хост сайта/admin
~~~
После авторизации вы можете добавить новые фильмы, режиссёров и жанры в базу данных для начала работы.
